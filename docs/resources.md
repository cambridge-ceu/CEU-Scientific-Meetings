[^VPN]: To check for the active status, ping from a CLI (Linux/Windows MS-DOS prompt, invoke with Windows key + r, type in "cmd" in the box and hit the return key) and issue 
    >
    > ping CMPCxxx.medschl.private.cam.ac.uk
    >
